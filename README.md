# Kanban-Board
A fully functional, drag-and-drop Kanban Board built with vanilla JavaScript. Features persistent local storage, responsive design, and a sleek dark UI.

## ✨ Features

- **Drag & Drop** - Move tasks between To Do, In Progress, and Done columns
- **Persistent Storage** - Tasks saved to localStorage (survives page reloads)
- **Add/Delete Tasks** - Create new tasks with title and description
- **Responsive Design** - Works on desktop and mobile devices
- **Dark Theme** - Modern dark UI with smooth animations

## 🛠️ Technologies

- HTML5
- CSS3 (CSS Variables, Flexbox, Transitions)
- Vanilla JavaScript (ES6+)
- LocalStorage API
- Drag and Drop API
